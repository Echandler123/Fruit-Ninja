# Fruit-Ninja
The game Fruit Ninja

This project is an interactive Fruit Ninja–style game that uses computer vision to let players slice fruit with real-time hand gestures. It was built in Python using OpenCV for video processing and MediaPipe for hand tracking. The game detects hand movement, spawns falling fruit, and uses collision detection to determine when fruit is sliced and update the score. It demonstrates concepts in computer vision, real-time rendering, and game loop design.
